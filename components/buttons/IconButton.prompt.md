Square icon-only button for toolbars, row actions, and compact controls. Always pass an accessible `label`.

```jsx
<IconButton label="Settings"><GearIcon/></IconButton>
<IconButton label="Run" variant="brand"><PlayIcon/></IconButton>
```

Variants: `ghost` (default), `solid` (white + border), `brand`. Sizes: `sm` / `md` / `lg`.
